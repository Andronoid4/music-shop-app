# 🎵 MusicShop - Музыкальный магазин

Курсовой проект: База данных музыкального магазина с веб-интерфейсом.

## 📋 Описание

Веб-приложение для управления музыкальным магазином с системой ролей пользователей, каталогом пластинок, корзиной покупок и административной панелью.

### Роли пользователей

| Роль | Права доступа |
|------|--------------|
| **Гость** | Только просмотр каталога |
| **Пользователь** | Просмотр + добавление в корзину + покупки |
| **Администратор** | Управление товарами (добавление, редактирование, удаление) |
| **Владелец** | Полный доступ + управление пользователями (блокировка) |

## 🚀 Запуск приложения

### Фронтенд

```bash
cd /workspace
npm install
npm run dev
```

Откройте браузер: **http://localhost:5173** (или другой порт, если 5173 занят)

### База данных

Выполните SQL скрипт для создания структуры БД:

```bash
psql -U postgres -f database.sql
```

## 🛠 Технологии

- **Frontend**: React 19, React Router DOM
- **Стили**: Tailwind CSS v4
- **Иконки**: Lucide React
- **Сборка**: Vite
- **База данных**: PostgreSQL

## 📦 Функционал

### Для всех пользователей
- ✅ Просмотр каталога пластинок
- ✅ Поиск и фильтрация
- ✅ Просмотр лидеров продаж

### Для зарегистрированных пользователей
- ✅ Добавление товаров в корзину
- ✅ Изменение количества товаров
- ✅ Оформление заказа

### Для администратора и владельца
- ✅ Добавление новых пластинок
- ✅ Редактирование существующих
- ✅ Удаление пластинок
- ✅ Загрузка обложек альбомов

### Только для владельца
- ✅ Просмотр списка пользователей
- ✅ Блокировка/разблокировка пользователей

## 🎨 Дизайн

- Современный минималистичный дизайн
- Адаптивная вёрстка (мобильные + десктоп)
- Интуитивная навигация
- Цветовая схема: синий акцент на светлом фоне
- Иконки для каждой секции

## 📸 Возможности

1. **Каталог товаров** - карточки с обложками, ценами, наличием
2. **Корзина** - управление количеством, подсчёт итоговой суммы
3. **Лидеры продаж** - топ-5 пластинок по продажам за год
4. **Админ-панель** - CRUD операции с товарами
5. **Управление пользователями** - таблица с блокировкой

## 🔐 Тестовые учётные данные

Для демонстрации выберите роль при входе:
- **Владелец** - полный доступ ко всем функциям
- **Администратор** - управление товарами
- **Пользователь** - покупки
- **Гость** - только просмотр
